Greetings Net-izen, this is the how-to page describing the construction of an HTML-CSS-Javascript Calculator. The calculator includes the basic operations of "addition," "subtraction," and so on, with even more.

<input type="text" id="firstNumber"> <input type="text" id="secondNumber">

These are the inputs, where one writes the numbers which the operators will do the math on(see below). This is what the code would look like individually. Type something!
<select id="mathTools"> <option value="+">+</option> <option value="-">-</option> <option value="*">*</option> <option value="/">/</option> <option value="^">^</option> <option value="√">√</option> </select>

This block of code is the dropdown, with all the operators, plus an exponential growth symbol and a square root symbol. Total of six operators and an infinitely possible amount of equations. This is what the code would look like individually. Click it!

<button type="button" id="solveButton> Click me for number!</button> <input type="text" id="answer"/>

This is the button to activate the calculator. After inputting two(or 1, in the case of √) numbers, and determining appropriate operator, the calculations would commence and you would get your answer. The answer will be written in a third input box(But do not write any numbers in it!).

To conclude and clarify, the reason these boxes are not working is because I did not link the javascript to them. If you would like to see the actual calculator at work, check it out in the "Calculator" page above. So thank you for checking this out! -Anthony
