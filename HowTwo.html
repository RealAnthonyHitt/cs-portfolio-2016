Greetings Net-izen, this is the how-to page describing the construction of an HTML-CSS-Javascript Calculator. The calculator includes the basic operations of "addition," "subtraction," and so on, with even more. 

            
        <input type="text" id="firstNumber">
        <input type="text" id="secondNumber">
        
These are the inputs, where one writes the numbers which the operators will do the math on(see below)

        
    <select id="mathTools">
          <option value="+">+</option>
          <option value="-">-</option>
          <option value="*">*</option>
          <option value="/">/</option>
          <option value="^">^</option>
          <option value="√">√</option>
        </select>
        
This block of code is the dropdown, with all the operators, plus an exponential growth symbol and a square root symbol.
